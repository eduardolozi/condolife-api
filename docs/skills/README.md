# Skills Catalog

This directory contains repository-level skill documents used by agents.

## Rules
- Skills are technical and reusable across modules.
- Module-specific behavior stays in `Identity/AGENTS.md` and `Condominiums/AGENTS.md`.
- Keep each skill concise and implementation-oriented.
- When adding a new skill, include: purpose, when to use, guardrails, and a short checklist.

## Available Skills
- `modular-monolith.md`
- `dotnet-pro.md`
- `dotnet-clean-arch.md`
- `efcore-postgres.md`
- `api-contracts.md`

## Suggested Usage
- Read the root `AGENTS.md` first.
- Read the module `AGENTS.md` second.
- Read only the skills needed for the current task.
